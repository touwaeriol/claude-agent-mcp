## 🚀 Quick Start

```bash
# Install
npm install claude-agent-mcp@1.1.0

# Use directly
npx claude-agent-mcp

# Install globally
npm install -g claude-agent-mcp

# Integrate with Claude Code
claude mcp add claude-agent-mcp --command npx --args claude-agent-mcp
```
